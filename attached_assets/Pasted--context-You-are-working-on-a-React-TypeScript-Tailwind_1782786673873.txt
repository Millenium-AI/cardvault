<design_goals>
The current table should stop feeling like a heavy spreadsheet.

Instead, make it feel like a premium, well-edited record list:
- Use softer borders or hairline dividers instead of strong block borders.
- Reduce the visual weight of the header.
- Increase whitespace just enough to breathe.
- Make important fields pop subtly.
- Make less important fields recede.
- Make row actions appear on hover so the table feels cleaner.
- Make numbers easier to scan by using tabular alignment.
- Use slight shadows, rounded container treatment, and restrained hover states.
</design_goals>

<recommended_changes>
Apply these ideas where appropriate:

1. Table container
- Wrap the desktop table in a rounded container.
- Add a subtle ring or shadow.
- Reduce the harshness of the outer border.

2. Header styling
- Use smaller, more elegant header text.
- Add uppercase tracking where it helps.
- Make the header feel lighter and less boxed-in.
- Keep drag handle functionality, but make it visually quieter.

3. Row styling
- Use softer row separators.
- Add a restrained hover background.
- Add a subtle shadow or highlight on hover if tasteful.
- Make rows feel like premium list items rather than grid cells.

4. Column treatment
- Card name should remain the strongest visual anchor.
- Price and quantity should use tabular numbers.
- Secondary columns like game, total, and metadata should be visually quieter.

5. Action visibility
- Edit/Delete actions should be hidden by default and fade in on row hover.
- Keep keyboard and click accessibility intact.

6. Alignment
- Numeric columns should be right-aligned or otherwise consistently aligned for faster comparison.
- Remove the “blocky centered cell” feeling if possible.

7. Density
- Reduce padding slightly, but do not make the table cramped.
- Find a balance between compact and premium.

8. Subtle luxury cues
- Slightly better spacing rhythm
- Softer hover states
- Cleaner shadows
- More restrained borders
- More elegant typography
</recommended_changes>
